Trabalho final consumo de API



Esta aplicação tem como objetivo manipular um formulário com javaScript, capturando os dados dos inputs, transportando via POST para um servidor. Em seguida é utilizado metodo GET para trazer as informações montando uma tabela dinamica. Na tabela estão incluidos botões para editar e excluir os cadrastro. Na edição é utilizado o metodo GET para trazer os dados no formulário e o PUT para retornar o cadastro atualizado para o servidor. Por fim o botão delete utiliza o metodo DELETE para excluir o cadastro escolhido do servidor.
